# Checkpoints

Milestone snapshots of `graph.json`. The **live graph** is always `../graph.json` — update it continuously; copy here only at milestones.

| Checkpoint | When |
|------------|------|
| `v0_initial` | Before coding (seed graph) |
| `v1_architecture` | After data model and scaffold |
| `v2_mvp` | All core requirements work |
| `v3_pre_deploy` | Before production deploy |
| `v4_submission` | Final submission state |

Each checkpoint includes:

- `vN_name.json` — copy of the graph at that moment
- `vN_name.md` — what changed since the previous checkpoint and why
